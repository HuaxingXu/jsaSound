jsaSound for Interactive Sound Modeling

jsaSound helps interactive sound designers to provide sound users (game, web, application developers) with models that have a simple and consistent API exposing exactly the interaction the designer intends.

The Sound Model API:
play()
release()
stop()

setParam([name, number],val)
setParamNorm([name, number], val)

getParam([name, number], ["name", "type", "val", "normval", "min","max"]) // type can return "range" or "url"

getNumParams()
getAboutText();
